# Snake Game using Java

## Project Overview

This project is a classic Snake Game developed in Java as part of my internship at Slash Mark IT Solutions. The game is built using Java Swing and demonstrates game-loop concepts, rendering, keyboard controls, and collision detection.

## Features

* Snake movement using keyboard controls
* Food generation
* Score progression through snake growth
* Collision detection
* Game over handling
* Graphical user interface using Java Swing

## Technologies Used

* Java
* Java Swing
* AWT
* Timers

## Learning Outcomes

* Game loop implementation
* Event handling
* Collision detection
* GUI development
* Object-Oriented Programming
